# IT Support Forum

Đồ án môn học Chuyên đề ASP.NET  
Trường Đại học Trà Vinh

---

## Giới thiệu

IT Support Forum là website chia sẻ kiến thức công nghệ thông tin được xây dựng bằng ASP.NET Core MVC.  
Hệ thống cho phép người dùng đăng bài viết, upload hình ảnh, bình luận và tìm kiếm nội dung giống mô hình mạng xã hội mini.

---

## Chức năng chính

- Đăng ký / Đăng nhập
- Phân quyền Admin và User
- CRUD bài viết
- Upload hình ảnh
- Bình luận bài viết
- Tìm kiếm bài viết
- Responsive UI
- Giao diện Bootstrap 5

---
